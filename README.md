# Running locally

1. Install Ruby and Jukyll - [Select the Guide for your OS](https://jekyllrb.com/docs/installation/)
2. Clone this repo
3. Run `bundle install` in the project folder
4. Run `bundle exec jekyll serve --livereload` to start the dev server
5. Make some edits!
